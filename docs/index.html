scout

fast, terminal-native file explorer.
dual-pane. git-aware. syntax-highlighted.

$ brew install mirageglobe/tap/scout

or build from source

  • zero config
  • single binary
  • async non-blocking
  • macos + linux

preview

features

  • ▪
    dual-pane layout
    explorer and syntax-highlighted preview side by side; collapse to full-screen preview with tab
  • ▪
    git integration
    live M + ? ! badges per file and branch name in the status bar — no config, no plugin
  • ▪
    chroma highlighting
    hundreds of languages, rendered in your current colour theme, in the preview pane
  • ▪
    time-aware themes
    nine themes auto-selected by hour of day; press t to cycle manually
  • ▪
    system stats
    live CPU, memory, directory size, and clock always visible in the header
  • ▪
    async non-blocking
    directory reads, git queries, and file watch run off the main loop — UI stays responsive
  • ▪
    editor handoff
    press e to open the file in $EDITOR; TUI suspends and resumes cleanly on exit
  • ▪
    mouse support
    click to select, scroll wheel in explorer and preview panes

keybindings

navigate
j / k move down / up
← / → parent dir / enter or preview
g / G jump to top / bottom
tab collapse / expand file pane
actions
e open in $EDITOR
o open with system default
i toggle hidden files
t cycle colour theme
/ search in preview
? help overlay
q quit

install

homebrew recommended 
tap the formula
brew tap mirageglobe/tap

install
brew install mirageglobe/tap/scout

upgrade later
brew upgrade mirageglobe/tap/scout
from source
clone and build
git clone https://github.com/mirageglobe/scout.git
cd scout
make build

run
./scout

requires go 1.21+
